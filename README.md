# Swift Transportation

Swift Transportation, now operating as Knight-Swift Transportation Holdings (KNX), is the largest full truckload motor carrier in North America, providing truckload, LTL, logistics, and intermodal transportation services across the US, Mexico, and Canada.

- **Website:** https://knight-swift.com
- **Knight Transportation:** https://www.knighttrans.com
- **Investor Relations:** https://investor.knight-swift.com

## Technology Integration

Knight-Swift Transportation does not offer a public REST API developer portal. Technology integration is primarily available through:

- **EDI (Electronic Data Interchange):** Standard ANSI X12 EDI transactions for freight tendering, shipment status, and invoicing
- **Load Board:** Digital freight marketplace for carriers and shippers
- **Customer Portal:** Online shipping tools for customers

## EDI Transaction Types

| EDI Transaction | Description |
|---|---|
| 204 | Motor Carrier Load Tender |
| 210 | Freight Invoice |
| 214 | Transportation Carrier Shipment Status Message |
| 990 | Response to a Load Tender |
| 810 | Invoice |
| 850 | Purchase Order |
| 856 | Ship Notice/Manifest |
